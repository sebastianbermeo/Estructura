  • ¿Qué es una pila y cola en PHP?

    Las pilas y colas en php son una lista doblemente enlazada, en ambas direcciones. Son bastante sencillas de usar. Seguro que muchos programadores de php usan arrays que permiten "simular" una pila y una cola ya que hay funciones que permiten sacar el último y primer elemento de un array (aparte de usar el índice 0 para el primer índice y el índice del tamaño del array menos 1 para el último elemento, suponiendo que no sea un array asociativo).
