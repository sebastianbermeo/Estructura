  • ¿Qué es una cola en C#?

    Una cola se utiliza para representar una colección de objetos primero en entrar, primero en salir (FIFO). Se utiliza cuando necesita el acceso de los elementos por orden de entrada, primero en salir. Es el tipo de colección no genérico que se define en el System.Collectionsespacio de nombres. Se utiliza para crear una colección dinámica que crece según la necesidad de su programa. En Cola, puede almacenar elementos del mismo tipo y de diferentes tipos. Generalmente, una cola es útil cuando accede a esa información de la misma manera en que se almacenó en la colección y es un almacenamiento temporal para almacenar datos.

¿Cómo crear la cola? en C#:

La clase de cola tiene cuatro constructores que se utilizan para crear la cola, que son los siguientes:

1. Queue(): este constructor se usa para crear una instancia de la clase Queue que está vacía y tiene la capacidad inicia

2. Cola (ICollection): este constructor se usa para crear una instancia de la clase Queue que contiene elementos copiados de la colección especificada, tiene la misma capacidad inicial que el número de elementos copiados y usa el factor de crecimiento predeterminado.

3. Queue (Int32): este constructor se usa para crear una instancia de la clase Queue que está vacía y tiene una capacidad inicial especificada, y usa el factor de crecimiento predeterminado.

4. Queue (Int32, Single): este constructor se usa para crear una instancia de la clase Queue que está vacía y tiene una capacidad inicial especificada, y usa el factor de crecimiento especificado.

¿Cómo eliminar elementos de la cola?

En Cola, puede eliminar elementos de la cola. La clase Queue proporciona dos métodos diferentes para eliminar elementos y los métodos son:

1. Borrar : este método se utiliza para eliminar los objetos de la cola.

2. Dequeue : este método elimina el elemento inicial de la cola.

¿Cómo obtener el elemento más alto de la cola?

En Queue, puede encontrar fácilmente el elemento superior de la cola utilizando los siguientes métodos proporcionados por la clase Queue:

1. Peek : este método devuelve el objeto al principio de la cola sin eliminarlo.

2. Retirar de la cola : este método devuelve el objeto al principio de la cola con una modificación que significa que este método elimina el elemento superior de la cola.

¿Cómo comprobar la disponibilidad de elementos en la cola?

En Queue, puede verificar si el elemento dado está presente o no usando el método Contain(). O en otras palabras, si desea buscar un elemento en la cola dada, use el método Contains() .
